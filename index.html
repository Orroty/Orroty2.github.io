Домашние работы 1 - 2

Первые два задания были объединены в 1 ссылку в связи с тем, что они затрагивали работу над одним сайтом. 1 ДЗ требовало призвести верстку сайта с использованием HTML.

Второе задание требовало дополнить сайт, созданный на первом задании каскадной таблицей стилей (CSS).

Домашняя работа 3

Третье задание требует:

  1. Работу с существующим макетом сайта созданым на сайте FIGMA
  2. Разбить ранее открытый макет на блоки (как в семинаре)
  3. Создать свой HTML сайт по блокам полученным в пункте 2.
  4. Добавить CSS без применения позиционирования.

Результат:

Результат разбиения на блоки.

Домашние работы 4-5

Четвертое задание требует:

  1. Задать стилистику сайту в соответствии с макетом
  2. Сделать позиционирование в сайте в соответствии с макетом
  3. Для позиционирования желательно использовать Flex

Результат:

Почти все позиционирование не сайте сделано с использованием Flexbox. Уточнение: значки в навигационной панели становяться оранжевыми при наведении.

Пятое задание требует:

  1. Создать сайт в соответствии с макетом
  2. Сделать стилистику и позиционирование в сайте в соответствии с макетом
  3. Для позиционирования желательно использовать Grid

Результат:

Почти все позиционирование не сайте сделано с использованием Grid и Flex. Уточнение: значки в навигационной панели становяться оранжевыми при наведении. Добавлен шрифт.

Домашние работы 6-7-8-9

Шестое-седьмое-Восьмое задания требуют:

  1. Полностью реализовать вёрстку проекта
  2. Добавить весь контент из макета
  3. Ко всем элементам контента добавить классы
  4. Добавить стили для всего контента сайта
  5. Расставить элементы в соответствии с макетом
  6. Создать планшетную версию проекта (768-1024px)
  7. Создать мобильную версию проекта (320-767px)
  8. Добавляем весь контент подвала сайта
  9. Подправляем версии сайта
  10. Добавляем эффекты наведения
